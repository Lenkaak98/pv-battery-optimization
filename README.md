# PV + Battery Optimization System
This project optimizes energy dispatch in a small household with a pv and battery storage system using PuLP and PvLib.

## Key Features
* *Solar Modeling:* Uses pvlib to estimate generation based on weather data.
* *Optimization:* Mixed Interger Linear programming via PuLP to minimize annual electricity costs.
* *Visualization:* Generates cost comparisons with and without battery as well as imported electricity with and without battery

Below are the sample results generated by the model:
* *Cost Comparison:* View the savings between a standard PV system and one with battery storage in the plots/ folder.
* *Electricity Import:* Visualizes how the battery reduces reliance on the grid during peak hours in the plots/folder.

## Data Sources
* *Weather Data:* Typical Meteorological Year (TMY) data used for PV generation modeling.
* *Load:* Extracted from Load Profile Generator,LPG (An open source tool to generate load profiles for different households)
* *Pricing:* Electricity tariff used is a Flat rate to calculate cost savings.
  
## Possible Improvements
* *Machine Learning:* Explore machine learning models for short term forecasting of demand and renewable generation to enable predictive battery dispatch
* *Degradation:* Extend battery model to include degradation and cycle life tracking
